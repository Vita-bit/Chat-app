Read a previously uploaded file from disk and send its base64-encoded content to the requesting user’s socket.

Query all chats the given user belongs to and send the list to that user’s socket.

server.get_users(requesting_user: str)[source]¶

Retrieve all registered usernames (excluding the requester) and send them to the requester’s socket.

server.handle_client(conn: socket, addr: tuple)[source]¶

Handle all communication with a single connected client for its entire session.

Manages authentication (login/register) on the first message, then dispatches subsequent messages to the appropriate handler functions. Cleans up the client entry on disconnection.

server.msg(user: str, content: str)[source]¶

Store a text message from user in the database and broadcast it to all members of the user’s currently active chat.

server.open_chat(chat_id: int, username: str)[source]¶

Validate the user’s membership in the chat, load the last 50 messages, and send them to the user’s socket. Also records the user’s active chat.

server.receive_file(user: str, file_name: str, file_data_b64: str, chat_id: int)[source]¶

Decode and save an uploaded file to disk, record it in the database, and notify all members of the chat.
